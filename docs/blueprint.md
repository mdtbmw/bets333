# **App Name**: Intuition Bet

## Core Features:

- Marketplace of Predictions: Display a dynamic feed of real-world event predictions with key details like the prediction question, live pool balance, participation stats, and countdown.
- Betting Arena: Allow users to connect their wallet, choose a side (YES/NO), and stake their $TRUST tokens, showing potential payouts and the growing pool size.
- Result Reveal: Utilize a verified external data source (Oracle) to fetch final results, automatically verifying and redistributing winnings via smart contracts.
- Trust Economy: Implement a Trust Score system where users gain/lose points based on prediction accuracy, influencing their leaderboard position and access to exclusive events.
- Community Engine: Enable users to join category-based communities, follow others' predictions, and earn badges for milestones, fostering social interaction and competition.
- Trending predictions insights: Use a generative AI model to provide short summaries or insights about why a prediction is trending; use reasoning to act as a 'tool' to incorporate only valid information.
- Admin Realm: Empower administrators to efficiently manage prediction events, user accounts, and platform configurations via a secure and intuitive admin interface.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5), evoking trust and intelligence.
- Background color: Light blue-gray (#ECEFF1), offering a clean, professional backdrop.
- Accent color: A vibrant purple (#9C27B0), emphasizing key interactions.
- Body and headline font: 'Inter', sans-serif, providing a modern, neutral aesthetic suitable for both headlines and body text.
- Use clean, geometric icons that represent prediction categories and community features.
- Employ a card-based layout for prediction events, ensuring a visually engaging and responsive user experience.
- Incorporate subtle animations for event updates and result reveals to enhance user engagement.
